## Props

| Prop       | Type       | Default     | Description                                                |
| ---------- | ---------- | ----------- | ---------------------------------------------------------- |
| `delay`    | `number`   | `500`       | The delay in milliseconds before the function is executed. |
| `callback` | `function` | `undefined` | The function to be executed after the delay.               |
