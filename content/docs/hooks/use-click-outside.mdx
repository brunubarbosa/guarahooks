## Props

| Prop             | Type        | Default | Description                                                  |
| ---------------- | ----------- | ------- | ------------------------------------------------------------ |
| `ref`            | `RefObject` | `null`  | The reference to the element to detect clicks outside of     |
| `onClickOutside` | `function`  | `null`  | The function to call when a click outside the element occurs |
